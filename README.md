# рҹ§  SMADS вҖ” Social Media Addiction Detection System

> **AI-Powered Digital Wellbeing Platform** | B.E. Final Year Project

An intelligent system that tracks, analyzes, and provides early warnings for unhealthy social media consumption patterns using Machine Learning and the Modified Bergen Social Media Addiction Scale (BSMAS).

| Layer          | Technology                                    |
| -------------- | --------------------------------------------- |

## рҹҸғ Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- MongoDB 7+ (running locally or Atlas URI)

### Setup
```bash
# Clone and navigate
cd SMADS

# Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python seed_data.py

# Frontend (new terminal)
cd frontend
npm install

# Start both
cd ..
start.bat
```

## рҹ“Ҡ Key Features

1. **Usage Data Ingestion** вҖ” High-frequency tracking of screen time, sessions, and app-switching
2. **Addiction Detection Engine** вҖ” ML-based classification (Normal / At-Risk / Addicted)
3. **Real-time Warning System** вҖ” Automated alerts when safety thresholds are breached
4. **Longitudinal Tracking** вҖ” Daily, weekly, monthly trends with recovery progress
5. **Smart Interventions** вҖ” Digital Detox suggestions and Focus Mode triggers
